# ⚡ 联系表单快速设置（5分钟完成）

## 🎯 最简单的方法：使用 Formspree

### 1️⃣ 注册 Formspree（2分钟）

1. 访问：https://formspree.io
2. 点击 "Sign Up" 注册（免费）
3. 验证邮箱

### 2️⃣ 创建表单（1分钟）

1. 登录后，点击 **"New Form"**
2. 表单名称：`iSperm Contact Form`
3. 接收邮箱：`market@isperm.com`
4. 点击 **"Create Form"**

### 3️⃣ 获取表单 ID（30秒）

创建后，您会看到表单 URL，例如：
```
https://formspree.io/f/abc123xyz
```

**复制 `abc123xyz` 这部分**（这就是您的表单 ID）

### 4️⃣ 配置到 GitHub（1分钟）

1. 打开您的 GitHub 仓库
2. 点击 **Settings** > **Secrets and variables** > **Actions**
3. 点击 **New repository secret**
4. 填写：
   - **Name**: `NEXT_PUBLIC_FORMSPREE_ID`
   - **Value**: 粘贴您的表单 ID（例如：`abc123xyz`）
5. 点击 **Add secret**

### 5️⃣ 完成！✅

1. 代码已经修改好了，会自动使用 Formspree
2. 下次推送代码时会自动部署
3. 部署完成后测试联系表单即可

## 🔄 或者使用 Web3Forms（完全免费，无需注册）

### 1️⃣ 获取 Access Key（1分钟）

1. 访问：https://web3forms.com
2. 输入邮箱：`market@isperm.com`
3. 点击 **"Get Your Access Key"**
4. 复制生成的 Access Key

### 2️⃣ 配置到 GitHub（1分钟）

1. 打开 GitHub 仓库
2. **Settings** > **Secrets and variables** > **Actions**
3. **New repository secret**
4. 填写：
   - **Name**: `NEXT_PUBLIC_WEB3FORMS_KEY`
   - **Value**: 粘贴您的 Access Key
5. **Add secret**

### 3️⃣ 完成！✅

代码会自动使用 Web3Forms（如果没有配置 Formspree）

---

## 📝 本地开发（可选）

如果想在本地测试，创建 `.env.local` 文件：

```env
NEXT_PUBLIC_FORMSPREE_ID=your_form_id_here
```

或

```env
NEXT_PUBLIC_WEB3FORMS_KEY=your_access_key_here
```

然后重启开发服务器。
