## 🆘 如果遇到问题

### 问题 1: 表单提交失败

**检查**：
- GitHub Secrets 中的名称是否正确（区分大小写）
- 表单 ID 是否正确（`meeejqka`）
- 浏览器控制台是否有错误信息
- Formspree 后台是否有提交记录

### 问题 2: 没有收到邮件

**检查**：
- Formspree 后台的提交记录
- 垃圾邮件文件夹
- Formspree 设置中的接收邮箱是否正确（应该是 `market@isperm.com`）

### 问题 3: 环境变量没有生效

**检查**：
- Secret 名称必须是 `NEXT_PUBLIC_FORMSPREE_ID`（完全匹配）
- 是否已重新部署（修改 Secrets 后需要重新触发部署）
- 可以在 Actions 日志中查看环境变量是否正确注入
